# lmk — React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

> Migrated from Create React App, which was [officially sunset in February 2025](https://react.dev/blog/2025/02/14/sunsetting-create-react-app). To scaffold a new Vite app from scratch: `npm create vite@latest`

## Getting started

```bash
npm install
npm run dev      # http://localhost:5173
```

## Available scripts

- `npm run dev` — start the dev server
- `npm run build` — type-check and build for production
- `npm run preview` — preview the production build locally

## Plugins

This project uses [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) which uses [Babel](https://babeljs.io/) for Fast Refresh.
